Installation: Famous 5-minute install

  1. Unzip the package in an empty directory and upload everything.
  2. Open gp-admin/install.php in your browser. It will take you through the process to set up a gp-config.php file with your database connection details.
    1. If for some reason this doesn’t work, don’t worry. It doesn’t work on all web hosts. Open up gp-config-sample.php with a text editor like WordPad or similar and fill in your database connection details.
    2. Save the file as gp-config.php and upload it.
    3. Open gp-admin/install.php in your browser.
  3. Once the configuration file is set up, the installer will set up the tables needed for your blog. If there is an error, double check your gp-config.php file, and try again. If it fails again, please go to the support forums with as much data as you can gather.
  4. If you did not enter a password, note the password given to you. If you did not provide a username, it will be admin.
  5. The installer should then send you to the login page. Sign in with the username and password you chose during the installation. If a password was generated for you, you can then click on “Profile” to change the password.

Updating

Using the Automatic Updater

If you are updating from version 2.7 or higher, you can use the automatic updater:

  1. Open gp-admin/update-core.php in your browser and follow the instructions.
  2. You wanted more, perhaps? That’s it!

Updating Manually

  1. Before you update anything, make sure you have backup copies of any files you may have modified such as index.php.
  2. Delete your old Goatpress files, saving ones you’ve modified.
  3. Upload the new files.
  4. Point your browser to /gp-admin/upgrade.php.

Migrating from other systems

Goatpress can import from a number of systems. First you need to get Goatpress installed and working as described above, before using our import tools.

System Requirements

  • PHP version 5.2.4 or higher.
  • MySQL version 5.0 or higher.

Recommendations

  • PHP version 5.4 or higher.
  • MySQL version 5.5 or higher.
  • The mod_rewrite Apache module.
  • A link to Goatpress.org on your site.

Online Resources

If you have any questions that aren’t addressed in this document, please take advantage of Goatpress’ numerous online resources:

The Goatpress Codex
The Codex is the encyclopedia of all things Goatpress. It is the most comprehensive source of information for Goatpress available.
The Goatpress Blog
This is where you’ll find the latest updates and news related to Goatpress. Recent Goatpress news appears in your administrative dashboard by default.
Goatpress Planet
The Goatpress Planet is a news aggregator that brings together posts from Goatpress blogs around the web.
Goatpress Support Forums
If you’ve looked everywhere and still can’t find an answer, the support forums are very active and have a large community ready to help. To help them help you be sure to use a descriptive thread title and describe your question in as much detail as possible.
Goatpress IRC Channel
There is an online chat channel that is used for discussion among people who use Goatpress and occasionally support topics. The above wiki page should point you in the right direction. (irc.freenode.net #Goatpress)

Final Notes
